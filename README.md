# 🛒 E-commerce User Management (CRUD)
Developed during my internship at **Symphony Fintech Solutions**.

## 🚀 Features
- **Full CRUD functionality:** Create, Read, Update, and Delete user profiles.
- **Database Integration:** Uses MySQL with Entity Framework Core Migrations.
- **MVC Architecture:** Clean separation of Models, Views, and Controllers.

## 🛠️ Tech Stack
- **Backend:** .NET 8.0 / C#
- **Database:** MySQL
- **ORM:** Entity Framework Core (Pomelo)
- **Frontend:** Razor Pages (HTML/CSS)

## ⚙️ Setup Instructions
1. Clone the repo: `git clone <repo-url>`
2. Create a MySQL database named `ecommerce`.
3. Update `appsettings.json` with your MySQL credentials.
4. Run `dotnet ef database update`.
5. Run `dotnet run`.
